### 社区生鲜团购系统+Vue3+SpringBoot
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、VSCode
技术栈：Java、MySQL、SpringBoot、Mybatis/MybatisPlus、Vue3、ElementUI、JavaScript、CSS

系统功能模块：
本系统包含管理员、用户、团长3个角色。
管理员角色：
1. 用户管理：管理系统中所有用户的信息，包括添加、删除和修改用户。
2. 配置管理：管理系统配置参数，如上传图片的路径等。
3. 权限管理：分配和管理不同角色的权限。
4. 商品信息管理：管理商品信息，包括新增、查看、修改、删除商品信息。
5. 商品分类管理：管理商品分类，包括新增、查看、修改、删除商品分类。
6. 自提商品管理：管理自提商品信息，包括接货信息和取货信息。
7. 商品资讯管理：管理商品资讯，包括新增、查看、修改、删除商品资讯。
8. 订单管理：管理商品订单，包括查看、删除不同状态的订单。
用户角色：
1. 商品信息管理：查看商品信息。
2. 商品资讯管理：查看商品资讯。
3. 购物车管理：管理购物车，包括新增、查看、修改、删除购物车中的商品。
4. 订单管理：查看和管理自己的订单，包括未支付、已取消、已支付、已退款、已完成等状态的订单。
5. 我的收藏管理：查看和管理自己的收藏。
6. 自提商品管理：查看和管理自提商品信息，包括接货信息和取货信息。
团长角色：
1. 订单管理：管理商品订单，包括查看、删除、核销不同状态的订单。
2. 自提点申请管理：管理自提点申请，包括新增、查看、修改、删除自提点申请。
3. 自提商品管理：管理自提商品信息，包括新增、查看、修改、删除接货信息和取货信息。
4. 商品信息管理：管理商品信息，包括新增、查看、修改、删除商品信息，并查看评论。
5. 商品资讯管理：查看商品资讯。
6. 购物车管理：管理购物车，包括新增、查看、修改、删除购物车中的商品。
